Explain how recursion can be used to solve problems.


Recursion

Essentially breaking down a problem into smaller parts by having a function call itself.

We need a base case to tell the function to stop calling itself, otherwise the program will crash.

Recursion is very fun and you should always use it if possible.



Apply recursion in Python.


Applying Recursion

So imagine a function defined by: def CountToN(n):

Inside the function, we simply do if n > 0: print(CountToN(n-1)), else: return 0

This function goes calls itself until it reaches zero, then returns zero. This then goes up a level, printing out 0, 1, 2, 3, 4, ..., n-1.

Define stack data structure.


Stacks

A stack is like a list, except you can only interact with the item at the 'top'. You can 'push' something to put it on the top of the stack, and 'pop' to take something from the top of the stack.

This behaviour is commonly referred to as Last In First Out.

Stacks are often used to reverse a list of items, or to turn a recursive algorithm into an iterative one, as well as memory allocation.

Explain the fundamentals of selection and iteration.


Selection

Essentially an If-Else Statement. It's essentially a decision, if a statement meets a certain criterion, the block of code will run, else, it won't run and optionally, a different block of code will run instead.

Iteration

Essentially a For or While Statement. Essentially, a block of code will loop, and depending on the type of statement used, it will terminiate after a set amount of iterations (For), or until a criterion is no longer true (While).

Lists

Lists are collections of items. They are usually mutable (Can be Changed), and in Python can be comprised of any data type.

In Python, they are defined by: Lister = ["dog", 1, 1.0232, False].

Tables

Tables are simply Lists of lists.

They are defined in Python by: Tabler = [[1, 1, 1], [1, 1, 1], [1, 1, 1]]

Strings

Strings are collections of characters. They are usually immutable, and contain pretty much any unicode compliant character within them.

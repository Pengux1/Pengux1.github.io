Get familiar with trees and graphs.


What is a Graph?

A graph is a collection of vertices V and a collection of edges E. Each edge connects a pair of vertices in the graph. Vertices are also sometimes referred to as nodes. Edges are sometimes referred to as arcs.

The two endpoints of an edge are said to be adjacent.

Paths, Trails and Walks

A Walk is a sequence of verticies where each pair of consecutive vertices in the walk are connected by an edge. The first vertext in the sequence is called the Start Vertex, and the last vertext in the sequence is called the End Vertex.

A Trail is a walk with no repeated edges.

A Path is a walk with no repeated edges and no repeated vertices (Except in the case of a Cycle which is a path that begins and ends on the same vertex.

A Circuit is a trail where the start vertex and the end vertex are the same.

Types of Graphs

A Directed Graph is a grpah in which edges have directions. Note that each edge can have one or two directions.

A Loop is an edge with the same vertex at each end. If there is more than one edge connecting a pair of verticies, these edges are called Multiple Edges.

A Simple Graph is a graph with no loops and no multiple edges. A Non-Simple Graph is a graph that is not simple. Duh.

A Connected Graph is a graph in which there is at least one path between any pair of verticies.

A Weighted Graph is a graph where each edge has an associated weight. Weights are usually a real number. For exammple, weights can represent distances between verticies, or the cost of transporting something between verticies. A Labelled Graph has edges and/or verticies that are labelled.

Trees

A Tree is a simple connected graph that has no cycles. A Spanning Tree of a graph G is a tree that contains a subset of the edges of G and all the vertices of G.
